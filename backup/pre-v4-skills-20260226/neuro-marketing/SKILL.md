# 行銷策略與神經科學

核心作者：Martin Lindstrom（Buyology）| 概念數：6 | 基礎：fMRI + SST 腦掃描研究，2,081名受試者

## 概念速查

1. **反效果警告效應** ⭐⭐⭐⭐⭐ — 健康警語反而刺激大腦渴望區(伏隔核)，負面訊息可能產生正面效果
2. **鏡像神經元消費效應** ⭐⭐⭐⭐⭐ — 看到別人用=自己想擁有，KOL行銷的科學基礎
3. **敘事整合式產品置入** ⭐⭐⭐⭐⭐ — 品牌融入故事=有效，壁紙式曝光=浪費錢
4. **腦掃描預測法** ⭐⭐⭐⭐⭐ — fMRI比問卷準確，品牌效應能「改寫」味覺體驗（百事vs可口可樂）
5. **吸血鬼效應** ⭐⭐⭐⭐ — 性感內容吸走品牌注意力，記住身體忘記品牌
6. **恐懼行銷驅動力** ⭐⭐⭐⭐ — 適度恐懼+明確解決方案=最強銷售組合，過度恐懼反而關閉反應

## 行銷檢查清單
- [ ] 行銷訊息在潛意識層是正面還是反效果？
- [ ] 有吸血鬼效應嗎（注意力被非核心元素吸走）？
- [ ] 產品置入是「深度整合」還是「壁紙式」？
- [ ] 是否利用鏡像神經元（展示使用場景）？
- [ ] 恐懼程度適中嗎？有配套解決方案嗎？
- [ ] 是否用行為數據而非問卷驗證效果？
