# 消費決策心理學

核心作者：Daniel Kahneman + Robert Cialdini + Martin Lindstrom | 概念數：6

## 概念速查

1. **雙系統思維框架** ⭐⭐⭐⭐⭐ — 快思直覺(S1) vs 慢想理性(S2)，大部分決策由S1自動完成
2. **聯想記憶與直覺思維** ⭐⭐⭐⭐ — S1的核心，自動構建世界的連貫解釋
3. **系統間相互影響** ⭐⭐⭐⭐ — S1和S2相互影響，非獨立運作
4. **互惠原理** ⭐⭐⭐⭐⭐ — 給予→義務→回報，人類獨特的債務循環系統
5. **自動化反應機制** ⭐⭐⭐⭐ — 互惠反應是無意識的「咔嚓-嗡嗡」自動反應
6. **潛意識購買驅力** ⭐⭐⭐⭐⭐ — 85%購買決策在意識之前已完成，傳統市調不可靠

## 決策檢查清單
- [ ] 是否過度依賴S1直覺判斷？
- [ ] 需要啟動S2進行深度思考嗎？
- [ ] 判斷是否受聯想記憶影響？
- [ ] 是否存在互惠的心理債務？
- [ ] 消費者「說」的和「做」的一致嗎？
