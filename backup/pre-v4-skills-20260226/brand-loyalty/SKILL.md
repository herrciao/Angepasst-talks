# 品牌建設與忠誠

核心作者：Martin Lindstrom（Buyology）| 概念數：5 | 基礎：fMRI + SST 腦掃描研究，2,081名受試者

## 概念速查

1. **體感標記決策捷徑** ⭐⭐⭐⭐⭐ — 大腦用過去經驗形成的情感捷徑做品牌判斷，純理性決策不存在
2. **儀式黏性法則** ⭐⭐⭐⭐⭐ — Oreo扭舔泡/Corona檸檬/Guinness慢倒，儀式讓品牌不可替代
3. **品牌宗教十大支柱** ⭐⭐⭐⭐⭐ — Apple粉絲腦反應=宗教信徒，10大要素：願景、敵人、神秘感、傳教精神...
4. **潛意識品牌暗示** ⭐⭐⭐⭐⭐ — 不需LOGO也能被辨識=終極品牌，無LOGO暗示比LOGO更強
5. **多感官品牌法則** ⭐⭐⭐⭐⭐ — 嗅覺+聽覺>視覺，多感官記憶度高70%

## 品牌建設檢查清單
- [ ] 品牌有獨特的消費儀式嗎？
- [ ] 具備宗教十大支柱中至少5項？
- [ ] 去掉LOGO後消費者還能辨識嗎？
- [ ] 品牌調動了幾種感官？（目標：至少3種）
- [ ] 每次品牌接觸是累積還是消耗體感標記？
