# 創新與第一性思維

核心作者：Ozan Varol（Think Like a Rocket Scientist）| 概念數：6

## 概念速查

1. **火箭科學家思維框架** ⭐⭐⭐⭐⭐ — 將不可能變為可能，將失敗轉化為勝利，約束轉化為優勢
2. **面對未知的勇氣** ⭐⭐⭐⭐ — 承認未知和不確定性的勇氣是創新突破的起點
3. **視角轉換能力** ⭐⭐⭐⭐⭐ — 想像不可想像的事物，從全新角度審視問題
4. **系統性思考方法** ⭐⭐⭐⭐ — 遵循嚴格思維過程，建立可重複的問題解決方法論
5. **失敗學習哲學** ⭐⭐⭐⭐ — 每次失敗=寶貴數據點，通往成功的必經之路
6. **第一性原理思考** ⭐⭐⭐⭐⭐ — 回到基本原理，質疑所有假設，拒絕「一直以來的做法」

## 創新檢查清單
- [ ] 質疑了所有基本假設嗎？
- [ ] 從全新視角審視問題了嗎？
- [ ] 將失敗視為學習機會嗎？
- [ ] 運用了系統性思考嗎？
- [ ] 有勇氣面對未知因素嗎？
