# 創業者領導力與企業成長

核心作者：Stefan Merath（Der Weg zum erfolgreichen Unternehmer + Dein Wille geschehe）| 概念數：12

## 概念速查

1. **專業人才 vs 企業家區分** ⭐⭐⭐⭐⭐ — 90%的創業者其實是專業人才在做老闆，只有轉型為真正的企業家才能突破
2. **瓶頸集中策略 (EKS)** ⭐⭐⭐⭐⭐ — 找到「一個」核心瓶頸並集中所有資源解決它，而非同時處理多個問題
3. **時間革命原則** ⭐⭐⭐⭐⭐ — 企業家的核心問題是時間分配：從「為企業工作」轉為「在企業上工作」
4. **信念系統識別與改變** ⭐⭐⭐⭐⭐ — 限制性信念（如「我必須對一切負責」）是成長最大的障礙，可以也必須被改變
5. **三大領導元模型** ⭐⭐⭐⭐⭐ — 戰略領導、願景領導、文化領導是企業家的三重角色
6. **控制圈-影響圈-宇宙圈** ⭐⭐⭐⭐ — 只在可直接控制的領域下手，不浪費精力在無法影響的事情上
7. **員工需求與發展** ⭐⭐⭐⭐ — 員工不滿是系統問題而非個人問題，需要主動傾聽和結構性解決
8. **從失敗中學習的三種態度** ⭐⭐⭐⭐ — 躺平型/怪罪外部型/反思行動型，只有第三種能成長
9. **願景的三重功能** ⭐⭐⭐⭐ — 方向感 + 困難時的能量來源 + 提升效率
10. **地位遊戲與Beta化** ⭐⭐⭐⭐ — 領導者必須理解團隊中的地位動態和Alpha測試
11. **七大領導行動領域** ⭐⭐⭐⭐ — 企業家領導的完整框架，應貼在辦公室牆上
12. **承諾的價值** ⭐⭐⭐⭐ — 外部承諾和社群壓力是推動改變的關鍵力量

## 企業家自檢清單
- [ ] 我是在「為企業工作」還是「在企業上工作」？
- [ ] 我能否識別出企業當前的一個核心瓶頸？
- [ ] 哪些限制性信念在阻礙我成長？
- [ ] 我的員工真正不滿意的是什麼？（有沒有問過他們？）
- [ ] 我的企業願景清晰嗎？團隊能複述嗎？
- [ ] 面對失敗我是哪種態度？
